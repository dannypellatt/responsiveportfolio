Deep Dive


Project Brief:

A multifunctional travel application aiding visits to Britain for Americans, and America for the British.

The purpose of this app is to act as a guide to the small differences between the UK and the US. It includes a translation dictionary, live weather searchable by city, a temperature converter, and a tipping calculator. It was the final project piece for my bootcamp.

Technology Used:

ASP .Net Core MVC || C# || MySQL || RestAPIs || HTML || CSS || Bootstrap || JQuery || JSON || Dapper || RestSharp || Postman

Design:

I chose a fun and cartoony theme, and a color scheme that fit with the red, white and blue flags. I used free-to-use icons, manipulated with photoshop for graphics. I worked with a graphic designer, who advised tweaks to improve readability, layout and styling.

Metrics & Accolades:

This project is featured by my bootcamp in their portfolio page, highlighting 10 of the best final projects made by past students.

View Code

||

Video Presentation

Deep Dive


Project Brief

In order to practice using REST APIs, create an application that simulates a conversation between Kanye West and Ron Swanson.

Features

This application call two different API's, one that returns random Kanye West quotes as a JSON object, and another that returns Ron Swanson quotes as a JSON array. The NewtonSoft.JSON package parsed these into strings. LINQ was used to trim and replace letters and a delay was inserted, and I concetenated in the correct punctuation, making the display look like a real online conversation.

Technology Used

ASP .Net Core || C# || RestAPIs || RapidAPI || Github || JSON || NewtonSoft || LINQ

Design

As this was a console application, the design revolved around delaying and manipulating the lines with C#.

Project Brief

A website designed to advertise a local beer/music festival in England, as well as provide information to festival-goers on the day.

Requirements

A simple site that will load easily in a place with spotty cell service. The logo, a brief summary, festival line-up and more information was given by client. It was to be used primarily on cell phones.

Technology Used

HTML || CSS || Bootstrap

Design

I created this website from scratch, as I wanted to be sure that there was no unnecessary code. It's completely responsive and designed for mobile first. The festival overview information comes before in-depth details, to keep bounce rate low for those just looking.
